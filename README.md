# Active Directory Home Lab (Windows Server 2022)

## Overview
This project demonstrates a hands-on Active Directory home lab built using Windows Server 2022 and a Windows 11 Pro client machine inside a VirtualBox environment. The lab simulates how enterprise organizations centrally manage users, computers, security policies, and shared resources using Active Directory Domain Services (AD DS).

A Windows Server 2022 machine was configured as a Domain Controller and a Windows 11 Pro client was joined to the domain in order to test authentication, Group Policy enforcement, and access to shared network resources.

## Lab Environment
- VirtualBox
- Windows Server 2022 (Domain Controller)
- Windows 11 Pro (Domain Client)
- Active Directory Domain Services
- Group Policy Management Console
- Windows File Services

## Key Concepts Demonstrated
- Active Directory Domain Services (AD DS)
- Organizational Unit (OU) design
- User and group management
- Domain-joined client machines
- Group Policy Objects (GPOs)
- Password and account lockout policies
- Network drive mapping
- File sharing permissions
